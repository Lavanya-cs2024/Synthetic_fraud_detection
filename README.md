### Problem statement
Fintech platforms enable fast onboarding but are highly vulnerable to fraud due to low entry barriers. Unauthorized users can create fake identities, 
leading to financial risks, while manual verification is not scalable for effective detection.

### Features
- Detects fraudulent transactions using ML models
- Behavioral profiling using clustering techniques
- Real-time prediction capability
- Clean and structured dataset processing
- Performance evaluation using multiple metrics

### Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Google Colab

### Machine Learning Models
- Logistic Regression
- Random Forest
- Support Vector Machine (SVM)
- K-Means Clustering (for behavioral profiling)

### Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC Curve

### How to Run
- git clone https://github.com/Lavanya-cs2024/Synthetic_fraud_detection.git
- Open the notebook in Google Colab
- Install required libraries (if needed)
- pip install -r requirements.txt
- Run all cells to train and test the model

### Results
The model successfully identifies fraudulent patterns with high accuracy and improved detection rate compared to traditional methods.
